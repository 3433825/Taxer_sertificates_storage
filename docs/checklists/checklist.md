# ✅ Чек-лист перевірок: Сховище сертифікатів

Цей документ містить узагальнений перелік функціональних областей та перевірок для сервісу [taxer.ua](https://taxer.ua).

## 1. Стартова сторінка
- [ ] Відображення кнопки для додавання сертифікату.
- [ ] Відображення текст-заглушки ("Нема жодного сертифікату") при порожньому списку.
- [ ] Відображення списку сертифікатів після завантаження.
- [ ] Розташування елементів відповідно до макету.
- [ ] Наявність індикатора активного запису (►) у списку.

## 2. Навігація та Екран завантаження
- [ ] Перехід на екран завантаження та повернення кнопкою «Назад».
- [ ] Відповідність тексту підказок та кнопок макету (мова, назви).
- [ ] Наявність та робота кнопки вибору файлу через системний діалог.
- [ ] Візуальна реакція зони drag&drop на наведення файлу.

## 3. Операції з сертифікатами
- [ ] Успішне завантаження валідного `.cer` файлу (Drag&Drop / Діалог).
- [ ] Обробка файлів невірного формату (`.txt`, `.jpg`) або пошкоджених файлів.
- [ ] Поведінка при повторному завантаженні існуючого сертифікату.

## 4. Панель деталей
- [ ] Відображення панелі при кліку на сертифікат.
- [ ] Відповідність назв та порядку полів: *Common Name → Issuer CN → Valid From → Valid To*.
- [ ] Оновлення даних при переключенні між записами.
- [ ] Відображення індикатора для прострочених сертифікатів.

## 5. Збереження даних (LocalStorage)
- [ ] Збереження даних після перезавантаження сторінки (F5) або закриття вкладки.
- [ ] Коректна поведінка при ручному очищенні LocalStorage.
